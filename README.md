# SPACE-WAR
A PC game, also known as a computer game is a type of video game played on a personal computer. We have developed a video game titled Space War. It consists of a spaceship, aliens and little space stations. The space ship is the main character and the aliens are the opponents. The space ship has to save the space stations from the horde of aliens by shooting bullets at the aliens. If the space ship fails to save any of the space stations, the lifespan of the spaceship decreases. If the spaceship saves the space station within its lifespan, the player wins, if not they lose.
